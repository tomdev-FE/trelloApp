# Trello App

Technology use `NextJS`, `Zustand`, `React-redux`, `material-ui`, `framer-motion`

## Getting Started

First, run the development server:

```bash
npm install
# or
yarn
```
After that, run:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
